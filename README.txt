1. EasyPark
此软件为基于智能手机的停车场寻车系统android客户端（不包含计步器部分）。
基于智能手机的停车场寻车系统以能够让用户在复杂布局的室内停车场快速找
到车辆为核心功能，前期可根据对用户停车习惯的数据分析（大数据技术）挖
掘出用户最常去的若干个停车场，或者结合百度地图提供“搜附近停车场”
功能供用户选择所要停车的停车场，同时在该部分会提供实时精准的车位信息
和路径导航。后期在停车场可以标识停车位的位置（立柱等）部署二维码的条
件下，可实现停车场内扫描二维码即定位、寻车路径规划、实时寻车路径导航
等功能，方便用户在复杂布局的停车场中快速找寻车辆。一方面为城市静态交
通规划提供数据支持，另一方面，通过解决停车和寻车问题从而提高城市整体效率。

3. PedometerExperimentServer
此软件为计步器实验服务器端，用于数据采集分析
