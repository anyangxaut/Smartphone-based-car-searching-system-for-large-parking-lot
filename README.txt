1.引言
    近年来，汽车越来越多地走进普通百姓家庭，汽车消费时代的悄然来临相应地带动了停车行业的兴起与发展，城市里“停车难，寻车难”问题也日趋严重。为了解决“停车难”的问题，众多的室内停车场纷纷出现，并且其建设规模日益扩大。然而，室内停车场在设计和使用上均比较复杂，伴随着停放车辆的增多，使得停车场内部“寻车难”的问题更加严重。由于室内停车场无法接收到GPS定位信号，手机信号也较弱，因此无法利用GPS技术和网络技术实现室内定位与导航功能。尤其对于大型室内停车场，由于其内部结构复杂、环境及标志物类似、位置直观性差等因素，使得车主很难快速寻找到自己的车辆，从而给车主造成很多困扰和不便，与此同时也变相造成了停车场内部的拥堵。
    随着移动通信及其服务技术的发展，智能手机在我们的生活中随处可见，为我们的工作和生活提供了诸多便利。同时，二维码作为一种全新的信息存储、传递和识别技术，已经在很多领域得到了广泛应用。目前，通过智能手机识别二维码信息已经成为移动设备的基本功能之一。
    针对上述问题，开发了一套基于智能手机的停车场寻车系统。该系统前期可实现停车功能，后期在停车场内部实现精准寻车功能。停车功能具有查询周边停车场实时车位信息、路径导航、智能推荐等特性；寻车功能具有部署简单、成本低、无需访问网络、无需使用GPS等优点。该系统为用户带来全新的智能化体验，使得“寻车难”不再成为社会生活的严重问题。
2.系统概述
    基于智能手机的停车场寻车系统以能够让用户在复杂布局的室内停车场快速找到车辆为核心功能，前期可根据对用户停车习惯的数据分析（大数据技术）挖掘出用户最常去的若干个停车场，或者结合百度地图提供“搜附近停车场”功能供用户选择所要停车的停车场，同时在该部分会提供实时精准的车位信息和路径导航。后期在停车场可以标识停车位的位置（立柱等）部署二维码的条件下，可实现停车场内扫描二维码即定位、寻车路径规划、实时寻车路径导航等功能，方便用户在复杂布局的停车场中快速找寻车辆。一方面为城市静态交通规划提供数据支持，另一方面，通过解决停车和寻车问题从而提高城市整体效率。
    基于智能手机的停车场寻车系统主要包括：应用程序端和服务器端两个主要部分。其中，应用程序主要分停车和寻车两大功能模块；服务器端包含数据存储，数据挖掘，停车日志管理，用户管理和消息推送等功能。
停车模块可分为5个功能部分：注册与登录，智能推荐，附近停车场，精准的车位信息，停车场路径导航。为驾车一族提供了便捷、准确、可靠的停车服务。
    寻车模块可分为标识位置信息的二维码子模块和基于智能手机的停车场寻车APP子模块。标识位置信息的二维码模块按照二维码数据格式将停车场信息、楼层信息和车位信息存储在二维码中，为后续扫码定位提供数据基础。基于智能手机的停车场寻车APP模块可以分为5个功能部分：停车场离线地图，扫描二维码定位，最佳寻车路径规划，实时寻车路径导航和查询停车历史记录。用户通过智能手机扫描识别二维码信息，结合寻车APP便可以实现复杂布局环境下的快速寻车。
